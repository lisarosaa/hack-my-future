## How to write in Markdown


- Italic: _text_
- Bold: **text**
- Header: #Header 1 ##Header 2
- Link: [text](link)
- reference link: [text][link]
- Images link:![text](link)
- reference images:![text][link]
- Block-quotes: >"quote."
- Lists: 
	- unordered (* text)
	- ordered (1. text; 2. text)
- Paragraphs:(space)
